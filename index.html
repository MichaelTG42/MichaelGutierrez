ASP.NET MVC Form Application

CISP – 238 Server-Side Scripting I

MIKES STORE was a final project application I built for a Server-Side Scripting course. The course objective was to build a windows web form application in the ASP.NET MVC framework. The application I built was called “MIKES STORE” named after myself. The project focused more on understanding the ASP.NET MVC framework rather than the aesthetics of the web page. The application was built on a three-tier architecture including the main index page which opens the application to the web browser, as well as category and subcategory pages to display the web stores fictitious products. Both category and subcategory pages then linked to an items page to provide and image and detailed information. The application also included a results, checkout, login, and user account page. Lastly, the project was connected to a Microsoft SQL Server Express database (The database did not actually hold any data. The requirements were to just link a database to the project.).

HTML/CSS/JS Web Application

CISP – 222 Web Page Applications

Angular Pro was the final project I created for a Web Page Applications course. The overall project required five web pages each keeping a consistent and aesthetic look and feel. The project required the use of HTML, CSS, CSS.mini, JavaScript, Multimedia links, Social media links, Site maps, and robots’ files. I chose to design the website as an online store that sold fishing tackle and lures. The site opens at the main index page which displays the stores featured products. The main index has an expandable hamburger tab on the top left-hand side of the page, below the header, that provides the user with categories linked to the specified category’s page. Each products image when hovered over by the cursor displays links to the items and checkout pages. The header (navigation) bar contains three links at the top right-hand side of the page that includes Home, About Us, and Shopping cart button.
